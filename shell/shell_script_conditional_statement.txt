shellの条件分岐:
条件分岐の形式
-条件処理は一つなら:
if 条件式 ; then
  処理
fi

簡単に言うと:
もし 条件式 ; なら
  処理
終了

-条件処理は二つなら:
if 条件式 ; then
  処理1
else
  処理2
fi

elseを利用すると,それ以外の場合は処理を指定できます

-二つ以上のとき
if 条件式1 ; then
  処理1
elif 条件式2 ; then
  処理2
else
  処理3
fi

elifを利用すると、ブロックを複数追加することにより、さらに多くの分岐を作ることも可能になります。










-shellでは条件分岐作成するためにtestコマンドも利用できます。
尚：オプションにより細かく指定することが可能である。
testのオプション:
(-eq) → equal → 同等
(-ne) → not equal → 等しくない
(-lt) → less than → 未満
(-le) → less than or equal → 以下
(-gt) → greater than → より大きい
(-ge) → greater than or equal → 以上

そのほかのオプション：
ファイルの比較の際に
(-nt) は「ファイル1 は ファイル2」より新しいを意味し、
(-ot) は「ファイル1 は ファイル2」より古いを意味する。

尚: Ture = 0  ,  Flase = 1
例：
コマンド: a=5
コマンド: b=6
コマンド: test a = b ; echo $?
結果:1 (False)

コマンド:test 1 = 1 ; echo $?
結果:0(True)
