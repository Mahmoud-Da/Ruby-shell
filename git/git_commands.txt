- clone
ポジトリ内の特定のリポジトリまたはブランチのコピーを作成するために使用されます。
- branch
ブランチを作成、一覧表示、名前変更、および削除できます。
- checkout
gitブランチによって作成された異なるブランチ間の切り替え。
- log
Gitリポジトリ内のコミットの記録を表示します
- status
作業ディレクトリとステージング領域の状態を表示します。どの変更がステージングされているか、どのファイルがステージングされていないか、どのファイルがGitによって追跡されていないかを確認できます。
- add
作業ディレクトリの変更をステージング領域に追加します。次のコミットに特定のファイルへの更新を含めることをGitに通知します。
- commit
変更などをローカルリポジトリに保存します。
- diff
2つのコミット内のファイル間、または現在のリポジトリと以前のコミット間の違いを示します。
- push
ローカルGitリポジトリ内のローカルブランチからリモートリポジトリにコミットを送信（またはプッシュ）できます。
- pull
リモートリポジトリからローカルリポジトリへの変更（コミット）にアクセスするために使用されます。
- fetch
ローカルのgitに、元のメタデータ情報から最新のメタデータ情報を取得するように指示します。（ファイル転送は行いません）。
- merge
別のリポジトリからの変更を組み込み、1つのブランチから別のブランチに変更をマージするために手動で使用できます。
